# Stress Alerts

## Purpose

Stress alerts notify the user when acoustic or combined stress values exceed the configured threshold.

## Alert Sources

Alerts can be triggered by:

- acoustic stress from live telemetry
- combined stress from backend analysis
- linguistic/text-derived stress

## Cooldown

Alerts use a cooldown window to prevent repeated rapid notifications.

## Alert Effects

An emitted alert can:

- add an intervention card
- play a short chime
- activate a visual flash
- mark chart points as alert points

## Muting

The settings panel allows the user to mute the audio chime without disabling visual/intervention alerts.

## Dismissal

Dismissed alerts remain in state as acknowledged rather than being treated as active.
