# Reporting Export

## Purpose

Reporting export provides a way to generate session summaries and browser print/PDF output.

## Data Used

Reports can use:

- stress data points
- transcript lines
- average stress
- maximum stress
- active time
- total alerts

## Browser Print/PDF

The current PDF pathway uses the browser print dialog. Print-specific CSS hides interactive controls and adjusts colours/borders for report output.

## CSV Export

CSV export is scaffolded and should include session time series values when fully implemented.

## Privacy Note

Reports may contain sensitive transcript and stress data. Treat exported files as sensitive records.
